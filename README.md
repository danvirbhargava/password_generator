# Secure Password Generator

A modern, secure password generator application built with Python and PyQt6.

## Features

*   **Cryptographically Secure**: Uses Python's `secrets` module for generating passwords.
*   **Real-time Strength Meter**: Visual and textual feedback on password strength based on entropy.
*   **Customizable**: Adjust length (6-64 chars) and toggle character sets (Uppercase, Lowercase, Numbers, Symbols).
*   **Modern UI**: Clean interface with Dark Mode support for better visibility.
*   **Clipboard Integration**: One-click copy with visual feedback.

## Requirements

*   Python 3.x
*   PyQt6


## Usage

Run the application from the terminal:

```bash
python3 password_generator.py
```
